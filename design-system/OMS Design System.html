05VOICE

Тон и принципы.

// принцип 01 Короткие утверждения с точкой. Заголовок — одно ёмкое предложение, всегда заканчивается оранжевой акцентной точкой. Без многоточий и восклицаний.
// принцип 02 Метки // моноширинным. Eyebrow-метки, заголовки колонок, нумерация — всегда JetBrains Mono в верхнем регистре, с трекингом 0.06–0.16em.
// принцип 03 Тонкие линии, без теней. Все группировки — через 1px бордер цветом --line. Никаких box-shadow, glow, blur.
// принцип 04 Оранжевый — точечно. Акцент работает как маркер: точка, дот процесса, hover, цифра в стате. Не делает заливку и не используется как фон.
